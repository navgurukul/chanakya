NavGurukul Scholarship Test



Yeh test shuru hone se pehle kuch important cheezein padh lein. Yeh cheezein aapko test dene ke time bahot kaam aayengi.

Before the starting of the test, please read the following instructions. All these instructions will be very helpful when you will attempt the test.

  1. Poora test 1 hour ka hoga. Kisi shaant jagah pe jaa ke iss test to dijiye. Isse aap ache se saare saavalon ka javab de payenge.
  2. Test dete waqt apne paas ek notebook aur pen rakhein. Aap koi bhi rough work notebook mein kar sakte hain.
  3. Har question ka answer apne phone pe hi test dete hue daal dein.
  4. Aapko cheating karne ka mauke mil sakte hain, lekin humein aap pe bharosa hai ki aap cheating nahi karenge :)
  1. The entire test is for 1 hour. Please attempt this test in a silent place.
  2. Keep a notebook while giving the test with you. You can do the rough work in the notebook.
  3. Answer the question only on your phone.
  4. We trust you to attempt this test with honesty :)

Ready Carefully



A lot of students all across India answer the NavGurukul's test to go through the 1 year program and get high paying software engineering jobs. Though there is a great reward at the end, but you will have to do a lot of hard work to get that reward. We would love to teach every student irrespective of their educational qualifications but sadly we require some basic knowledge of certain things. We use this test to see if you have that basic knowledge.

Kafi sare students pure India se NavGurukul ka test dete ha taki woh NavGurukul ke 1 saal wale program ke madad se high paying software engineering jobs le sake. Hala ki akhir me bohut bada eenam hota ha, magar uske liye apko kafi jada mehnat bi karni padhti hai. Hume har kisi student ko padhna chate hai chae uske educational qualifications kuch bi ho magar hume kuch basic knowledge chaiye kuch chizo me. Yeh test ke duwara hum yeh jane ki kosish karte ha ki kya apke pass woh basic knowledge hai.

Some students try to cheat and score well in the test but sadly are not able to do well during the program. All of them have to leave the course.

Kuch students meh cheate karke score karlete ha but woh program meh ache tarike se perform ni kar pahte ha, Jiske karan uneh yeh program chorna padhta ha.

If you are proceeding with the test please make sure you do it yourself without anyone's help. We will disqualify you from NavGurukul if we detect any form of cheating.

Agar aap ish test ke liye aage badh rahe hai toh kripaya ish test ko khudse de bina kisike madad ke. Agar hume pata chalta ha ki apne cheating ki hai toh hum apko disqualify kardenge aage.
